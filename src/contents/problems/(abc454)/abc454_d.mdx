## 問題

`(`,`x`,`)` からなる文字列 $A,B$ が与えられる。\
$A$ へ下の操作を自由に行えるとき、$B$ に一致させることが可能か判定せよ。

- 部分文字列 `(xx)` を `xx` に置き換える
- 部分文字列 `xx` を `(xx)` に置き換える

<Border><Size s="13">*（マルチテストケース）* $\sum_{\mathtt{Testcases}} |A|+|B| \leq 2 \times 10^6$</Size></Border>

## 解法

- エスパーすると、$A, B$ にそれぞれ`(xx)`→`xx`の置換をできるだけ適用して\
同じ文字列になるか判定すればいいことが分かります
- 後でここ書き直すので許してください

## 実装

```cpp
void solve() {
    multicases([&](){
        string cin(a,b);

        // sから括弧をすべて取り除く変換を行う
        auto conv=[&](string s){

            // 空文字列tを用意し、
            // sから一つずつ文字を取り出してsを再構築する
            string t="";
            for(auto&c:s){
                t+=c;

                // 再構築中、tのsuffixが(xx)ならばxxに変換する
                if(t.length()>=4&&t.substr(t.length()-4,4)=="(xx)"){
                    rep(i,4)t.pop_back();
                    t+="xx";
                }
            }
            return t;
        };
        
        ansYn(conv(a)==conv(b));
    });
}
```
