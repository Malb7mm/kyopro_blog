## 問題

$N \times N$ のグリッドがある。\
左上のマスから始め、上下左右のマスへの移動を繰り返し、\
同じマスを2度通らずに右下のマスへたどり着く経路であって、\
マス $(A,B)$ のみを通らない経路があるか判定・構築せよ。\
<Border><Size s="13">*（マルチテストケース）* $N \leq 10^3, \enspace (A,B) \neq (1,1), (N,N)$</Size></Border>

## 解法

### 市松パリティで考える
- ゴールにたどり着くには\
「白→黒→白→黒→……→白」の順で移動する必要がある
  - つまり白マスが黒マスより一つ多い必要がある

- グリッドが偶数×偶数の場合は白マスと黒マスの数が同じ

### 具体的な経路の構築
- 縦二列ずつで割って、$B$ 列目の相方になる列を $C$ とする（赤枠）

- $B$ 列目を無視して考えると、蛇行するだけでゴールにたどり着ける
  - $B$ 列目に入るときは必ず白マスなので、\
  必ず $(A,B)$ を回避して下のような蛇行ができる

- $C$ 列目を進むとき、ついでに縦に蛇行して $B$ 列目のマスも踏む
  - $C$ 列目を進む向きは必ず下方向
  - $C$ が奇数列目か、偶数列目かで場合分けすると書きやすい
    - 奇数列目の場合は、$(1,C)$ から右→下→左→下→……の繰り返し
    - 偶数列目の場合は、$(1,C)$ から下→左→下→右→……の繰り返し
    - $A$ 行目だけ横移動をスキップ

## 実装

- [提出](https://atcoder.jp/contests/abc454/submissions/75105213)

```cpp
void solve() {
    multicases([&](){
        ll cin(n,a,b);
        if(n%2==1){cout<<"No"<<el;return;}
        if((a+b)%2==0){cout<<"No"<<el;return;}

        ostringstream oss;
        ll c=(b%2==0)?b-1:b+1;
        string v="D";

        rep(i,1,n+1){
            if(i==b){
                oss<<"R";
            }
            else if(i==c){
                if(c%2==1){
                    string h="R";
                    // 1スタート（jは横移動時の縦座標に合わせる）
                    rep(j,1,n){
                        if(j!=a)oss<<h,h=h=="R"?"L":"R";
                        oss<<"D";
                    }
                }
                if(c%2==0){
                    string h="L";
                    // 2スタート（jは横移動時の縦座標に合わせる）
                    rep(j,2,n+1){
                        oss<<"D";
                        if(j!=a)oss<<h,h=h=="R"?"L":"R";
                    }
                }
                oss<<"R";
                v=v=="D"?"U":"D";
            }
            else{
                rep(j,n-1)oss<<v;
                oss<<"R";
                v=v=="D"?"U":"D";
            }
        }
        auto ans=oss.str();
        ans.pop_back();//最後にRが入るので取り除く
        cout<<"Yes"<<el<<ans<<el;
    });
}
```
